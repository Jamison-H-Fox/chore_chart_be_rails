# What are we building together?
- We are building the back end for a simple chore chart application using ruby on rails.
- The application will allow users to create accounts, add chores, assign chores to users, and track the completion of chores.
- A few of the features requires for acceptance are:
    - Single user account (for the parent) with multiple child profiles nested under it.
    - Ability to add, edit, and delete chores.
    - Ability to assign chores to specific child profiles.
    - Ability for child profiles to mark chores as complete.
    - Ability to set chores as recurring (daily, weekly, monthly).
    - Ability to get a list of chores paginated by week.
